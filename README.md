## feature
- 支持基于 OKR 的个人和团队目标管理
- 支持灵活的任务管理
- 支持自动生成周报
- ……

## 快速使用
```bash
$ cd && git clone https://github.com/mkzilla/koala.git
$ # 请按需编辑编辑 koala/hack/docker/koala.json 这个配置文件
$ cd koala/hack/docker && sudo docker-compose up
$ # 访问 127.0.0.1 即可

```
